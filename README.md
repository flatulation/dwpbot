# dwpbot
dwpbot is a Discord bot where you claim bennies, bet and mug other users. 
It goes without saying this is just a joke, a fun bot.

## Run
```
git clone https://github.com/flatulation/dwpbot
```
Using yarn:
```
yarn
yarn build
yarn start
```
